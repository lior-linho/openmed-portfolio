# Standardized Vessel Models

This folder contains 3 standardized vessel geometries generated in Blender.

| Model | Geometry (.glb) | Centerline (.json) |
|--------|------------------|--------------------|
| Straight Vessel | `vessel_straight.glb` | `vessel_straight_centerline.json` |
| Curved Vessel | `vessel_curved.glb` | `vessel_curved_centerline.json` |
| Bifurcation Vessel | `vessel_bifurcation.glb` | `vessel_bifurcation_centerline.json` |

All models were procedurally generated in Blender using Python automation (`scripts/generate_vessels.py`).
